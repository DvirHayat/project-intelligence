# Project Intelligence

A comprehensive platform for intelligent project management and analytics.

## Project Structure

- **backend/** - Backend API services
- **frontend/** - Frontend application
- **analytics/** - Analytics and reporting services
- **infra/** - Infrastructure and deployment configurations
- **docs/** - Documentation

## Getting Started

### Prerequisites
- Docker and Docker Compose
- Python 3.9+
- Node.js 16+

### Running the Project

```bash
docker-compose up
```
